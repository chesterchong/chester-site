## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Personalizing

Search the codebase for `TODO(Chester)` to find the spots that still need your content:

- `src/app/page.js` — education, highlights, and past roles on the about page
- `src/app/components/ProjectSearch.js` — your projects
- `src/app/components/Footer.js` — social links
- `src/app/writing/` — add a folder with a `page.mdx` per post
- `src/app/layout.js` — your deployed domain
- `src/app/favicon.ico` / `favicon.dark.ico` — replace with your own icons
